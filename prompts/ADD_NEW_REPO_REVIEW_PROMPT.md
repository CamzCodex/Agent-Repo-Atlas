# Add New Repo Review Prompt

Review the candidate repo and produce a compact atlas entry.

Include:

- `id`
- `url`
- `category`
- `type`
- `aliases`
- `stack`
- `summary`
- `primaryUseCases`
- `agentUseInstruction`
- `acquisitionMode`
- `riskLevel`
- `riskFlags`
- `license`
- `maintenance`
- `relatedRepos`
- `reviewRequiredBeforeUse`
- `recommendedAgentAction`

Keep the record factual. Listing means classified, not approved.
